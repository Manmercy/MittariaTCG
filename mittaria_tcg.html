HOW TO PLAY

🎯 Overview

MITTARIA GENESIS เป็นเกม TCG แบบ 1v1 — ทำลาย Gate ทั้ง 5 ของฝ่ายตรงข้ามเพื่อชนะ แต่ละ Gate คือไพ่จาก Deck ที่วางคว่ำหน้าไว้เป็นชีวิต

🃏 Card Types

CHARACTERวาง Field, โจมตี, ป้องกัน

Stats: ATK / DEF / Cost / Crystal Value
เลือก Attack Position (ATK vs ATK) หรือ Guard Position (ATK vs DEF)

COMPANIONAttach กับ Character

ให้ buff หรือ ability — ตายพร้อม Character ที่ Attach อยู่

SPELLลงทันที, Response ได้

เล่นฟรีจากมือได้ทุก Phase

⚡ Turn Structure

① Draw Phase

จั่ว 2 ใบ (Turn แรกจั่ว 0) — ถ้า Deck หมด = แพ้ทันที — Hand Limit 7 ใบ

② Main Phase

ลง Character ได้ 1 ใบต่อ Turn (เลือก Position) — ลง Companion / Spell ได้ไม่จำกัด — Sacrifice Character จาก Field เพื่อ Crystal

③ Battle Phase

Character ที่อยู่ก่อน Turn นี้โจมตีได้ — Character ใหม่ยังโจมตีไม่ได้ (Summoning Sickness) — โจมตีได้เฉพาะ Lane เดียวกัน

④ End Phase

จบ Turn ฝ่ายตรงข้ามเริ่ม Turn

💎 Crystal Cost

จ่าย Crystal โดย Sacrifice Character จาก Field ออกไป — Lane นั้นจะว่าง

⚔️ Combat

สถานการณ์วัดผล
ชน Attack PositionATK vs ATKATK สูงกว่าชนะ, อีกฝ่ายตาย
ชน Guard PositionATK vs DEFถ้า ATK > DEF, รับตาย
เท่ากัน—Standoff ไม่มีใครตาย
Lane ว่าง—ทำลาย Gate → ไพ่นั้นขึ้นมือ

🏆 Win Conditions

  • Gate ทั้ง 5 ของฝ่ายตรงข้ามถูกทำลาย = ชนะ
  • Deck ของตัวเองหมดใน Draw Phase = แพ้
