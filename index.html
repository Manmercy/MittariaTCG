HOW TO PLAY

🎯 Overview

MITTARIA GENESIS เป็นเกม TCG แบบ 1v1 — ทำลาย Gate ทั้ง 5 ของฝ่ายตรงข้ามเพื่อชนะ แต่ละ Gate คือไพ่จาก Deck ที่วางคว่ำหน้าไว้เป็นชีวิต

🧩 Deck Building / วิธีจัด Deck

Deck Rule แบบสั้น

1 Deck = 40 ใบ โดยแนะนำให้จัดเป็น Character 24 / Companion 8 / Spell 8

  • Character ต้องเยอะที่สุด เพราะเป็นตัวลงสนาม, ป้องกัน Gate, และใช้จ่าย Cost
  • Companion ใช้เสริมตัวหลัก แต่ถ้าเยอะเกินจะมือเน่าเพราะไม่มีตัวให้ใส่
  • Spell ใช้แก้สถานการณ์ เช่น จั่ว, ย้ายเลน, ชุบชีวิต, ซ่อม Gate
  • Prototype นี้ให้ใส่การ์ดซ้ำได้สูงสุดประมาณ 3 ใบต่อชื่อการ์ด

Deck Building Rule / กติกาการจัด Deck

  • Deck ต้องมี 40 ใบ
  • แนะนำให้มี Character 24 ใบ เพื่อไม่ให้มือตันและมีตัวลงสนามพอ
  • แนะนำให้มี Companion 8 ใบ เพื่อเสริมตัวหลักโดยไม่แย่งพื้นที่ Character มากเกินไป
  • แนะนำให้มี Spell 8 ใบ เพื่อแก้สถานการณ์ เช่น ย้ายเลน, จั่ว, ชุบชีวิต, ซ่อม Gate
  • Prototype นี้ให้ใส่การ์ดซ้ำได้สูงสุดประมาณ 3 ใบต่อชื่อการ์ด

Recommended 40-card ratio

Character 60% / Companion 20% / Spell 20%
เหตุผล: เกมต้องมี Character มากพอให้ไม่ตัน, Companion พอให้เกิด combo, และ Spell พอให้มีทางแก้สถานการณ์

⚡ Turn Structure

① Draw Phase

จั่ว 2 ใบ (Turn แรกจั่ว 0) — ถ้า Deck หมด = แพ้ทันที — Hand Limit 7 ใบ

② Main Phase

ลง Character ได้ตามจำนวนเลนและ Cost ที่จ่ายได้ (เลือก Formation ตอนลง) — ลง Companion / Spell ได้ไม่จำกัด — Sacrifice Character จาก Field เพื่อจ่าย Cost

③ Battle Phase

Character ที่อยู่ก่อน Turn นี้โจมตีได้ — Character ใหม่ยังโจมตีไม่ได้ (Summoning Sickness) — โจมตีได้เฉพาะ Lane เดียวกัน

④ End Phase

จบ Turn ฝ่ายตรงข้ามเริ่ม Turn
